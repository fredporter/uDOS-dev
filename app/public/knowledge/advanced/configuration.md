# Configuration

Customize uMarkdown to match your preferences and workflow.

## Accessing Configuration

Click the **Config** button in the navigation menu to access the configuration panel.

## Available Settings

### Theme

Choose between:

- **Auto**: Follows your system preference
- **Light**: Always light mode
- **Dark**: Always dark mode

### Editor Settings

- Font family and size
- Line height
- Editor width
- Syntax highlighting

### File Locations

Configure where your files are stored and accessed.

## Configuration Files

uMarkdown uses several configuration files:

- `locations.json` - File system locations
- `settings.json` - App preferences
- `themes.json` - Theme customization

## Advanced Configuration

For power users, you can directly edit configuration files in the new Config panel, which provides:

- Code editor on the left for direct editing
- Settings controls on the right
- Quick save/restore functionality
- Multiple config file selection

## Tips

- Changes are saved automatically
- Invalid JSON will show an error
- You can reset to defaults anytime

Happy customizing! ⚙️
