# Welcome to uMarkdown

Welcome to **uMarkdown**, the next-generation markdown editor that brings your documents to life!

## What is uMarkdown?

uMarkdown is a powerful markdown editor that supports multiple formats:

- **Editor Mode**: Clean, distraction-free writing with live preview
- **Story Mode**: Create interactive typeform-style forms and surveys
- **uCode Mode**: Execute code blocks and build interactive experiences
- **Guide Mode**: Build knowledge bases and documentation
- **Config Mode**: Manage your app settings and configurations
- **Marp Mode**: Create beautiful presentations from markdown

## Getting Started

1. Choose a format from the menu
2. Start writing in markdown
3. Preview your content in real-time
4. Save and share your work
