## ðŸŽ¯ Task 9: RSS Feed Generation

**Mission:** Generate RSS 2.0 feeds from binder markdown files

### Architecture

**BinderFeed Class:**

- Scan folder for .md files
- Extract frontmatter (title, date, author)
- Generate feed items with content preview
- Support multiple formats (RSS 2.0, JSON Feed)
- Cache for performance

**Frontmatter Format:**

```markdown
---
title: Research Notes
date: 2026-01-17
author: Fred
tags: [research, data]
description: Latest findings
---

Content here...
```

**Feed Output:**

- `MyBinder/feed.xml` - Public RSS feed
- JSON format alternative
- Auto-update on content change

### Implementation Plan

**Phase 1: Metadata Extraction**

```python
extract_frontmatter(md_path: Path) -> Dict
extract_content_preview(content: str) -> str
```

**Phase 2: Feed Generation**

```python
class BinderFeed:
    def __init__(self, binder_path: Path)
    def scan_files(self) -> List[FeedItem]
    def generate_rss(self) -> str
    def generate_json(self) -> Dict
    def save_feed(self) -> Path
```
